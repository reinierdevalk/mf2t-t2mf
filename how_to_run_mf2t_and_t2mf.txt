=====================
Running mf2t and t2mf
=====================

mf2t and t2mf are Unix executables downloaded from http://mac.softpedia.com/get/Developer-Tools/MacMIDIt2mf.shtml 

After cd-ing into the folder that holds them, they can be run as follows.   

MIDI to text:
$ ./mf2t 'bla.mid' 'bla.txt'

Text to MIDI:
$ ./t2mf 'bla.txt' 'bla.mid'

For more information on the program see the README (taken from http://matsp888.no-ip.org/~mats/software/winunix/midi/mf2t/ )
